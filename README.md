# Movie App

Movie App is a responsive web application built with Vite, React, and Tailwind CSS. It leverages The Movie Database (TMDB) API to let users search for movies and view details such as posters, titles, ratings, and release datesâ€”all in a sleek, user-friendly interface.

## Features

- **Responsive Design:** Optimized for all devices.
- **Movie Search:** Quickly find movies using the TMDB API.
- **Detailed Information:** View movie posters, titles, ratings, and release dates.

## Technologies Used

- **Vite:** A fast, modern build tool.
- **React:** For building the user interface.
- **Tailwind CSS:** For streamlined, utility-first styling.
- **TMDB API:** To fetch real-time movie data.
